# 📖 Overview

**NexusPay** is a Business Intelligence project that analyzes digital payment transactions using **SQL** and **Power BI**. The dashboard transforms raw transactional data into meaningful insights, helping stakeholders monitor payment performance, customer activity, merchant operations, and revenue trends.

The project demonstrates the complete data analytics workflow—from SQL-based data preparation to interactive Power BI dashboard development.

---

# 🎯 Project Objectives

* Analyze digital payment transactions.
* Monitor revenue and payment trends.
* Evaluate customer transaction behavior.
* Track merchant performance.
* Measure payment method usage.
* Identify business growth opportunities.
* Support data-driven decision-making.
